ClueBot III
===========

ClueBot III is an automated bot running against Wikipedia EN (see https://en.wikipedia.org/wiki/User:ClueBot_III).

It is designed to archive pages, where the `{{User:ClueBot III/ArchiveThis}}` page is transcluded.
